# Php User Profile

## Description

Php code to edit and show in a new page, selected user data.

## Technologies

To develop this project, we use the php language, which permits to create scripts for dynamic web pages.

## How the code works

- When we open the <i>index.html</i> page, a form opens up to fill in your personal data.

- Once you press the button "Salva profilo", you will be sent to the second page which displays the selected data.
